#### 任务 1.3：游戏主循环（1.0h）⭐ 必做

| 字段 | 内容 |
| --- | --- |
| 任务 ID | T-1.3 |
| 优先级 | P0 |
| 预估工时 | 1.0h |
| 依赖 | T-1.2 |

**任务描述**：
- 使用 `requestAnimationFrame` 实现游戏主循环
- 引入固定时间步长（Fixed Timestep），逻辑更新 60Hz
- 分离 `update(dt)` 与 `render()`

**验收标准**：
- [ ] 控制台每秒输出约 60 次 tick
- [ ] 不同性能设备上逻辑速度一致

**关键代码片段**：
```ts
class Game {
  private accumulator = 0;
  private readonly step = 1000 / 60; // 60Hz 逻辑更新

  loop(timestamp: number) {
    const delta = timestamp - this.lastTime;
    this.accumulator += delta;
    while (this.accumulator >= this.step) {
      this.update(this.step);
      this.accumulator -= this.step;
    }
    this.render();
    requestAnimationFrame(t => this.loop(t));
  }
}
```

**关键代码**：
```ts
class Food {
  position: Point;

  spawn(snake: Snake) {
    const empty: Point[] = [];
    for (let x = 0; x < GRID.COLS; x++) {
      for (let y = 0; y < GRID.ROWS; y++) {
        if (!snake.body.some(s => s.x === x && s.y === y)) {
          empty.push({ x, y });
        }
      }
    }
    this.position = empty[Math.floor(Math.random() * empty.length)];
  }
}
```

```ts
export const storage = {
  get<T>(key: string, fallback: T): T {
    try {
      const v = localStorage.getItem(key);
      return v ? JSON.parse(v) : fallback;
    } catch { return fallback; }
  },
  set(key: string, value: unknown) {
    try { localStorage.setItem(key, JSON.stringify(value)); } catch {}
  },
};
```

**任务描述**：
- 监听 `touchstart` / `touchend`
- 计算滑动方向（水平/垂直，取距离更长的）
- 阈值：滑动 ≥ 30px 才触发

**验收标准**：
- [ ] 移动端上下左右滑动控制蛇移动
- [ ] 滑动方向不会误判（防误触）

**关键代码**：
```ts
let startX = 0, startY = 0;
canvas.addEventListener('touchstart', e => {
  startX = e.touches[0].clientX;
  startY = e.touches[0].clientY;
});
canvas.addEventListener('touchend', e => {
  const dx = e.changedTouches[0].clientX - startX;
  const dy = e.changedTouches[0].clientY - startY;
  if (Math.abs(dx) > Math.abs(dy)) {
    if (Math.abs(dx) > 30) inputDir = dx > 0 ? 'RIGHT' : 'LEFT';
  } else {
    if (Math.abs(dy) > 30) inputDir = dy > 0 ? 'DOWN' : 'UP';
  }
});
```
